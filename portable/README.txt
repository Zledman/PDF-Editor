# PDF Editor - Portable Version

## How to use
1. Ensure the computer has Java installed (Java 17 or later).
2. Double-click `start.bat`.
3. Wait for the window to say "Started PdfserverApplication".
4. Open your web browser and go to: http://localhost:8082

## Troubleshooting
- If the black window closes immediately, try opening a command prompt in this folder and running `java -jar pdf-editor.jar` to see the error message.
- If port 8082 is already taken, you may need to close other applications.
